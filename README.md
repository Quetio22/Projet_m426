# Projet WebApp de Messagerie

## Description du projet

Dans le cadre des modules **Développement Web M426, M295 et M294**, notre équipe développe une WebApp de messagerie sous forme de **Single Page Application (SPA)**.  

L’objectif du projet est de concevoir une application moderne, dynamique et intuitive permettant aux utilisateurs de communiquer en temps réel à travers une interface web fluide et responsive.

Ce dépôt principal regroupe l’ensemble des ressources, de la documentation et de l’organisation du projet. Il sert également de point central pour coordonner le développement entre les différents membres de l’équipe.

---

## Objectifs du projet

- Développer une application de messagerie moderne
- Mettre en place une architecture SPA performante
- Utiliser des technologies web actuuelles
- Travailler en équipe avec une organisation structurée
- Appliquer les bonnes pratiques de développement logiciel
- Gérer efficacement les versions et la collaboration via GitHub

---

## Organisation de notre équipe

Le projet est réalisé par une équipe de **4 personnes**.  

Une bonne organisation est essentielle afin d’optimiser la répartition des tâches, améliorer la communication et assurer un développement efficace.

Chaque membre contribue à différentes parties du projet afin de garantir une progression constante et une bonne coordination entre le frontend, le backend, la documentation et les tests.

Nous utilisons une méthode de travail collaborative avec :
- une répartition claire des tâches,
- un suivi de l’avancement du projet,
- l’utilisation de branches Git dédiées,
- des revues de code,
- et une documentation centralisée.
