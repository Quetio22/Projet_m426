## Description du projet : 
Le projet consiste à développer une application web permettant aux utilisateurs de communiquer entre eux via un système de messagerie en ligne.<br>
L’application inclura un frontend (interface utilisateur) et un backend (gestion des données, API, authentification), tous deux entièrement codés par notre équipe.<br>

Les utilisateurs pourront créer un compte, se connecter, envoyer et recevoir des messages, gérer leurs conversations et interagir avec d’autres membres de la plateforme.

##  Description de la construction du projet 

Ce dépôt regroupe l’ensemble du travail réalisé dans le cadre du module **M426**.  
Il inclut :

- Le backend  
- Le frontend  
- La documentation de développement (maquettes, user stories, exigences…)  

L’objectif est de fournir une architecture claire, maintenable et facilement extensible.
